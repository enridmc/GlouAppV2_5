## Vision funcional fijada

Glourant sera una plataforma para restaurantes centrada inicialmente en una demo funcional:

```text
Cliente escanea QR
-> abre sesion de mesa
-> ve carta
-> anade productos al carrito
-> envia pedido
-> el restaurante ve la comanda
-> cocina recibe el pedido
-> cocina cambia el estado
-> cliente ve la actualizacion
-> se simula pago o cierre manual
-> la mesa queda libre
```

La primera version no busca construir todo el SaaS completo. La prioridad es una demo solida, presentable y util para probar en un local real.

## Arquitectura acordada

La arquitectura final prevista se divide en tres sistemas de datos:

- **MySQL** como fuente de verdad para datos transaccionales: usuarios, restaurantes, locales, mesas, reservas, carta maestra, comandas, pagos, facturas, stock, caja y auditoria.
- **Firebase Firestore** mas adelante para tiempo real: estado vivo de mesas, pedidos activos, cocina, notificaciones, sesiones activas y actualizaciones instantaneas.
- **MongoDB** mas adelante para documentos flexibles: cartas publicadas, snapshots de pedidos, historico de actividad y analitica.

Para la primera fase se trabajara con:

```text
Flutter Web + API Backend + MySQL
```

Regla importante: Flutter no debe conectarse directamente a MySQL. La app debera consumir una API backend.

## Incidencias corregidas

Durante la ejecucion en Flutter Web aparecio una pantalla en blanco.

Primero se comprobo que:

- `flutter analyze` funcionaba correctamente;
- `flutter build web` funcionaba correctamente;
- el servidor web respondia con HTTP 200.

Despues aparecio un error real de layout:

```text
RenderFlex ... constraints: BoxConstraints(... h<=Infinity)
Cannot hit test a render box with no size
```

La causa estaba en la pantalla de login:

- habia un `Flex` horizontal dentro de un `SingleChildScrollView`;
- usaba `crossAxisAlignment: CrossAxisAlignment.stretch`;
- Flutter Web no podia calcular la altura porque el scroll entregaba altura infinita.

Se corrigio cambiando el layout del login para no estirar en un eje con altura infinita y usando `mainAxisSize: MainAxisSize.min` en el panel lateral.
